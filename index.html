About Me

Saya merupakan siswa yang sedang menempuh pendidikan di jurusan Rekayasa Perangkat Lunak (RPL) dan memiliki ketertarikan pada dunia teknologi informasi. Saya mulai mengenal pemrograman dan pembuatan website melalui pembelajaran di sekolah, seperti menggunakan HTML, CSS, serta dasar pengelolaan project dengan Git dan GitHub. Bagi saya, belajar coding adalah proses yang menantang sekaligus menyenangkan.

Dalam keseharian, saya berusaha untuk terus belajar dan memperbaiki kemampuan diri, baik secara teori maupun praktik. Saya juga terbiasa bekerja dengan tekun, tidak mudah menyerah, dan terbuka terhadap masukan. Ke depan, saya ingin mengembangkan kemampuan di bidang IT agar dapat menghasilkan karya yang bermanfaat dan menjadi bekal untuk melanjutkan pendidikan maupun dunia kerja.
